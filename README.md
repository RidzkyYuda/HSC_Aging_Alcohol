This repository contains R codes used for the analysis of single-cell RNA-seq datasets from sorted murine Lineage- Sca-1+ cKit+ (LSK) cells as described in Yuda et al, 2025 manuscript (https://www.biorxiv.org/content/10.64898/2025.12.25.696246v1). The codes are written in R programming language and posted as .Rmd files.
